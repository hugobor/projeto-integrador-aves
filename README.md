## Execução

### Linux
```bash
$ virtual env venv
$ pip install Flask
$ . venv/bin/activate
$ export FLASK_APP=avesapp
$ export FLASK_ENV=development
```

### Windows
*Não testei :(*
```CMD
> py -3 -m venv venv
> venv\Scripts\activate
> pip install Flask
> set FLASK_APP=avesapp
> set FLASK_ENV=development
> flask run
```

Então abra http://localhost:5000/ no navegador.
